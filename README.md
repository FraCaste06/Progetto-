# Progetto
# # Requisiti per l'accesso:
-interprete: Python 3.x
-Librerie standard utilizzate:
sqlite3

# # Istruzioni per eseguire il programma:
-compilazione: non è necessaria alcuna compilazione perchè il programma è scritto in Python e eseguito tramite interprete 
-installare Python 3.x
-aprire il terminale 
-entrare nella cartella contentente i file: main.py e database.sql
-eseguire il programma con il comando: python3 main.py o talvolta python main.py
